Zonenalarmsystem - Eventdokumentation 1.0

Dies ist die Event-Dokumenation für das Zonenalarmsystem. Weitere Erläuterungen zum Projekt können dem Domain Vision Statement entnommen werden. Alle Events halten sich im Aufbau an die hier festgelegte Struktur, folgend werden zur Übersichtlichkeit der Payload und Header getrennt.
